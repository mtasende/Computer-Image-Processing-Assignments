Antes de correr el script main.m se deberá editar y definir la variable "os" si usted utiliza el sistema operativo Windows. Los scripts fueron probados en Linux.
os = 'win' para Windows
os = 'lin' para Linux o cualquier otro SO tipo Unix

Luego se puede ejecutar main.m

NOTA: Dentro de ej2.m se puede descomentar "analizarBD" para que se cargue la base de datos de MUCT y se calculen los píxels de piel según las marcas manuales de la BD. Este proceso demora algunos minutos. El resultado se guarda en las variables "rPielFinal", "gPielFinal" y "bPielFinal", que son las utilizadas por ej2.m .
